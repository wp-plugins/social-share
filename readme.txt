== Description ==
Add Facebook/Twitter share buttons to your website.

== Installation ==
* Upload `SocialShare` folder to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
* Go to "Appearance", drag the widget into your primary widget area
* See http://www.jpreece.com/wordpress/social-share-plugin-for-wordpress/ for complete tutorial
